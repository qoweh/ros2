# docs/log 인덱스

`docs/log`는 개발 중간 기록, 시행착오, 실험 메모를 모아두는 곳이다.

## 1. 현재 핵심 로그

- `01_pingpong_rl_debug_log.md`
  - 최근 물리/보상/컨트롤러 디버깅 요약
- `02_curriculum_and_reward_plan.md`
  - staged curriculum, 자세 유지, 관절/부드러움 보상 설계

## 2. 기존 정리 문서 보관

- `help/`
  - 개념 설명, 시행 중간 설명 문서
- `step/`
  - 다음 단계 아이디어와 계약 문서

즉, 예전 `docs/help`와 `docs/step`은 삭제하지 않고 `docs/log` 아래로 옮겨서 history를 보존한다.
