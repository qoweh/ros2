요약
본 졸업작품은 ROS2와 Gazebo 환경에서 로봇팔을 구동하고, Flutter 앱 UI를 통해 제어하는 시스템을 구현하는 것을 목표로 한다.
주요 대상은 로봇 제어를 처음 학습하는 입문자 및 모바일 기반 로봇 제어 인터페이스를 필요로 하는 사용자이며, 시뮬레이션과 실제 제어 로직을 동시에 사용할 수 있도록 구성한다.
핵심 기능은
1. 앱 기반 수동 제어(관절/엔드이펙터 제어)
2. ROS2 통신 기반 상태 모니터링
3. 특정 작업(예: pick-and-place) 수행을 위한 로봇팔 실행
이다.
결과적으로 본 시스템은 단순 조작 도구를 넘어, 수동 조작과 학습 기반 자동 동작을 결합한 실용형 로봇 제어 플랫폼으로 활용될 수 있다.

⸻

연구방법 및 참고서
방법은
1. ROS2-Gazebo 기반 로봇팔 시뮬레이션 환경 구축
2. Flutter 앱과 ROS2 브리지 연동
3. 작업 수행 데이터 수집(시연/로그)
4. 학습 모델(모방학습 또는 강화학습) 적용
5. 작업 성공률·정확도·응답지연 평가
순서로 진행한다.
학습 단계에서는 pick-and-place 같은 목표 작업을 정의하고, 수동 조작 데이터 및 시뮬레이션 반복 학습을 통해 자동 수행 정책을 생성한다.
이후 앱에서 수동/자동 모드를 전환해 성능을 비교 검증한다.
